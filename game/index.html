Beat-Slay-Repeat — teste local

Abra esta página via servidor local (python -m http.server na pasta game) para áudio e caminhos relativos funcionarem corretamente.

Teste de verificarAcerto: isso não é o jogo final — só inventa uma “nota” na faixa da tecla A, na altura certa. Cada acerto consome essa nota; para testar de novo, clique outra vez em «Simular nota…» (o painel também zera score/reputação nesse clique).

Dica ritmo: acertos podem ficar mais baixos que a música; erro costuma ser mais alto para destacar. Ajuste no código: Entrada.definirVolumeSomAcerto(0.25), Entrada.definirVolumeSomErro(1).
